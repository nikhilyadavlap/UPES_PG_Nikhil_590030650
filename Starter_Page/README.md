# Installing Visual Studio Code

## What is Visual Studio Code?

Visual Studio Code, commonly called VS Code, is a free and lightweight code editor developed by Microsoft. It is widely used by students, developers, researchers, and data scientists for writing, editing, running, and managing code.

VS Code supports many programming languages such as Python, C, C++, Java, JavaScript, HTML, CSS, and Markdown. It also supports extensions, Git integration, terminal access, debugging, and project folder management.

For PG / M.Tech students, VS Code will be used for:

- Writing Python programs
- Writing C/C++ programs
- Creating GitHub repositories
- Editing Markdown files
- Managing research and project folders
- Creating portfolio websites
- Working with Git and GitHub
- Writing documentation for projects

# Installing Git on Windows

## What is Git?

Git is a distributed Version Control System (VCS) used to track changes in files and collaborate on software development projects. It allows developers and researchers to maintain the history of their work, manage multiple versions of code, and collaborate using platforms such as GitHub.

Throughout this course, Git will be used to:

- Track project changes
- Store code on GitHub
- Maintain research and project documentation
- Collaborate with classmates
- Build an online academic portfolio

---

# Verify Git Installation

Open **Git Bash**, **PowerShell**, or the **VS Code Terminal** and run:

```bash
git --version
```

Example output:

```text
git version 2.55.0.windows.2
```

---

# Configure Git

Set your username:

```bash
git config --global user.name "Your Name"
```

Set your email address:

```bash
git config --global user.email "your.email@example.com"
```

Verify the configuration:

```bash
git config --global --list
```

Example output:

```text
user.name=John Doe
user.email=john@example.com
credential.helper=manager
```

---

# Verify Git Credential Manager

Run:

```bash
git credential-manager version
```

A version number indicates that Git Credential Manager is installed correctly.

# Create Your First Repository

Create a new folder:

```bash
mkdir MyFirstRepo
cd MyFirstRepo
```

Initialize Git:

```bash
git init
```

Create a README file:

```bash
echo "# My First Repository" > README.md
```

Stage the file:

```bash
git add .
```

Create the first commit:

```bash
git commit -m "Initial commit"
```
