# Portfolio Website

This directory contains the source code for the GitHub Pages website of the **UPES_PG_Template** repository.

The portfolio is automatically published using **GitHub Pages** from the `docs/` directory.

## Live Website

🌐 https://vibhug0077.github.io/UPES_PG_Template/

---

## Directory Structure

```
docs/
├── index.html      # Main landing page
├── style.css       # Website styling
├── script.js       # JavaScript functionality
└── README.md       # Documentation
```

- HTML5
- CSS3

## Features

- Responsive landing page
- Modern portfolio layout

- Mobile-friendly design
- Lightweight static website

---

## Local Development

Clone the repository:

```bash
git clone https://github.com/vibhug0077/UPES_PG_Template.git
```

Navigate to the documentation folder:

```bash
cd UPES_PG_Template/docs
```

Open `index.html` in your browser or use a local development server such as the VS Code Live Server extension.

---

## Deployment

This website is deployed using **GitHub Pages**.

Repository Settings:

- **Branch:** `main`
- **Folder:** `/docs`

Any changes committed and pushed to the `docs` directory are automatically published.
